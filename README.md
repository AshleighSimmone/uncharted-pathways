# Uncharted Pathways 🌍

Uncharted Pathways is a travel website designed to inspire meaningful, immersive travel experiences beyond the typical tourist path.

This project is part of my journey into front-end development, where I’m building real-world projects while growing my skills in HTML, CSS, and eventually JavaScript.

---

## ✨ About the Brand

Uncharted Pathways is centered around the idea of building a bold life through travel — focusing on hidden gems, cultural connection, and intentional experiences rather than rushed itineraries.
